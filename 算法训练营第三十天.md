##  **322. 零钱兑换**  

[322. 零钱兑换 | 动态规划 | 完全背包 | 状态转移 | 代码随想录](https://programmercarl.com/0322.零钱兑换.html)

## 我的思路

看完题解自己来整理一下思路。

1.dp数组及下标的含义

dp[j]表示当总金额为j的时候硬币数量为dp[j]

2.递推公式

dp[j]=min(dp[j-coins[i]]+1,dp[j])

也就是减去当前硬币，之前能够用的最小值，再加1个当前硬币。和dp[j]做比较。

3.初始化

不能让初始值覆盖后面递推的公式所以dp[j]初始化为最大值。dp[0]=0表示当总金额为0的时候需要0个硬币

4.确定遍历顺序

排列：外层遍历背包，内层遍历物品

组合：外层遍历物品，内层遍历背包

物品无限，内层正序循环即可。

## 问题总结

**1.最大值和最小值的写法**

`INT_MAX / INT_MIN`

**2.dp的大小**

` vector<int>dp(amount+1,INT_MAX);`

因为金额要计算到amount，所以数组大小要开到amount+1

**3.内层循环的起始位置**

 `for(int i=0;i<coins.size();i++){`

​        `for(int j=coins[i];j<=amount;j++){`

两个原因：

1.要保证j能够装得下这个coins[i]，所以不从0开始

2.dp[j-coins[i]]如果从0开始数组下标会减成负的越界

**4.为什么要检查dp[j-coins[j]]是不是初始值**

因为外层遍历的是硬币，每次只用**当前这种硬币**去更新 dp 数组

所以 `dp[1]` 在 `coin=3` 那轮永远不会被碰到（因为 `j` 从 `coins[i]=3` 开始），而 `coin=5` 那轮同样跳过了 `j=1`。

```
class Solution {
public:
    int coinChange(vector<int>& coins, int amount) {
            vector<int>dp(amount+1,INT_MAX);
            dp[0]=0;
            for(int i=0;i<coins.size();i++){
                for(int j=coins[i];j<=amount;j++){
                   if (dp[j - coins[i]] != INT_MAX) { // 如果dp[j - coins[i]]是初始值则跳过
                    dp[j] = min(dp[j - coins[i]] + 1, dp[j]);
                }
                }
            }
            if(dp[amount]==INT_MAX)return -1;
            return dp[amount];
        
    }
};
```

2.虽然n不为0，但是后面的推导都需要0这个状态。

```
class Solution {
public:
    int numSquares(int n) {
        vector<int>dp(n+1,INT_MAX);
        dp[1]=1;
        dp[0]=0;
        vector<int>coins(101,1);
        for(int i=1;i<101;i++){
            coins[i]=i*i;
        }
        for(int i=1;i<=100;i++){
            for(int j=coins[i];j<=n;j++){
                
                    dp[j]=min(dp[j-coins[i]]+1,dp[j]);
                
            }
        }
        return dp[n];
    }
};
```

```
class Solution {
public:
    bool wordBreak(string s, vector<string>& wordDict) {
        unordered_set<string> wordSet(wordDict.begin(), wordDict.end());
        vector<bool> dp(s.size() + 1, false);
        dp[0] = true;
        for (int i = 1; i <= s.size(); i++) {   // 遍历背包
            for (int j = 0; j < i; j++) {       // 遍历物品
                string word = s.substr(j, i - j); //substr(起始位置，截取的个数)
                if (wordSet.find(word) != wordSet.end() && dp[j]) {
                    dp[i] = true;
                }
            }
        }
        return dp[s.size()];
    }
};
```



## 多重背包

[动态规划：关于多重背包，你该了解这些！ | 多重背包 | 动态规划 | 01背包 | 代码随想录](https://programmercarl.com/背包问题理论基础多重背包.html)

## 笔记

有N种物品和一个容量为V 的背包。第i种物品最多有Mi件可用，每件耗费的空间是Ci ，价值是Wi 。求解将哪些物品装入背包可使这些物品的耗费的空间 总和不超过背包容量，且价值总和最大。

多重背包和01背包是非常像的， 为什么和01背包像呢？

每件物品最多有Mi件可用，把Mi件摊开，其实就是一个01背包问题了。

如果物品数量很多的话，C++中，这种操作十分费时，主要消耗在vector的动态底层扩容上。（其实这里也可以优化，先把 所有物品数量都计算好，一起申请vector的空间。

这里也有另一种实现方式，就是把每种商品遍历的个数放在01背包里面在遍历一遍。

```
#include<iostream>
#include<vector>
using namespace std;
int main() {
    int bagWeight,n;
    cin >> bagWeight >> n;
    vector<int> weight(n, 0);
    vector<int> value(n, 0);
    vector<int> nums(n, 0);
    for (int i = 0; i < n; i++) cin >> weight[i];
    for (int i = 0; i < n; i++) cin >> value[i];
    for (int i = 0; i < n; i++) cin >> nums[i];

    vector<int> dp(bagWeight + 1, 0);

    for(int i = 0; i < n; i++) { // 遍历物品
        for(int j = bagWeight; j >= weight[i]; j--) { // 遍历背包容量
            // 以上为01背包，然后加一个遍历个数
            for (int k = 1; k <= nums[i] && (j - k * weight[i]) >= 0; k++) { // 遍历个数
                dp[j] = max(dp[j], dp[j - k * weight[i]] + k * value[i]);
            }
        }
    }

    cout << dp[bagWeight] << endl;
}
```
