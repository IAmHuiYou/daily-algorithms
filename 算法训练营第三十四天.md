复述思路

1.dp数组含义

dp[i]表示以nums[i]为结尾的最长递增子序列的长度

2.递推公式

dp[j]是i前面的任意一个值

若nums[j]<nums[i]

则dp[i]=dp[j]+1

需要遍历i前面每一个值，取最大的值

3.初始化

最长子序列长度最小为1，所以全部初始化为1

4.遍历顺序

从小到大

最终结果是遍历dp数组找出一个最大值。

## 我的代码

```
class Solution {
public:
    int lengthOfLIS(vector<int>& nums) {
        vector<int>dp(nums.size(),1);
        int result=1;
        for(int i=1;i<nums.size();i++){
            for(int j=0;j<i;j++){
                if(nums[i]>nums[j])
                dp[i]=max(dp[j]+1,dp[i]);
            }
            result=dp[i]>result?dp[i]:result;
        }
        return result;
    }
};
```

        vector<int>dp(nums.size(),1);

## 卡的思路

复述思路。

子数组指的是连续的子数列

1.dp含义

`dp[i][j]以i-1为结尾的nums1,以j-1为结尾的nums2的两个数组的最长重复子数组长度。`

2.递推公式

`若nums[i-1]==nums[j-1],dp[i][j]=dp[i-1][j-1]+1`

3.初始化

`dp[0][j]和dp[i][0]都是无意义的状态，全为0`

其他默认为0.

4.遍历顺序

两个数组从前往后两层for循环遍历即可。 

i和j的上限应该是等于nums.size()，因为表示i-1、j-1

最后把dp数组遍历一遍得到最大值。



## 我的代码

```
class Solution {
public:
    int findLength(vector<int>& nums1, vector<int>& nums2) {
        vector<vector<int>>dp(nums1.size()+1,vector<int>(nums2.size()+1,0));
        int result=0;
        for(int i=1;i<=nums1.size();i++){
            for(int j=1;j<=nums2.size();j++){
                if(nums1[i-1]==nums2[j-1])
                dp[i][j]=dp[i-1][j-1]+1;
                  if (dp[i][j] > result) result = dp[i][j];
            }
        }
        return result;
    }
};
```
