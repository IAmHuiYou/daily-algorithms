## 我的思路

## 卡的思路

1. 确定dp数组（dp table）以及下标的含义

dp[i][j] ：表示从（0 ，0）出发，到(i, j) 有dp[i][j]条不同的路径。->你看，别人是用到达每一个点的路径数作为dp，这样终点直接看dp就可以了

2.确定递推公式

想要求dp[i][j]，只能有两个方向来推导出来，即dp[i - 1][j] 和 dp[i][j - 1]。

此时在回顾一下 dp[i - 1][j] 表示啥，是从(0, 0)的位置到(i - 1, j)有几条路径，dp[i][j - 1]同理。

那么很自然，dp[i][j] = dp[i - 1][j] + dp[i][j - 1]，因为dp[i][j]只有这两个方向过来。

3.确定遍历顺序

这里要看一下递推公式dp[i][j] = dp[i - 1][j] + dp[i][j - 1]，dp[i][j]都是从其上方和左方推导而来，那么从左到右一层一层遍历就可以了。

这样就可以保证推导dp[i][j]的时候，dp[i - 1][j] 和 dp[i][j - 1]一定是有数值的。

第一行和第一列都只有一种路径，全部初始化后一行一行的赋值就行。

## 我的代码

```
class Solution {
public:
    int uniquePaths(int m, int n) {
        vector<vector<int>> dp(m,vector<int>(n,0));
        for(int i=0;i<n;i++)dp[0][i]=1;
        for(int j=0;j<m;j++)dp[j][0]=1;
        for(int i=1;i<m;i++){
            for(int j=1;j<n;j++){
                dp[i][j]=dp[i-1][j]+dp[i][j-1];
            }
        }
        return dp[m-1][n-1];
    }
};
```

class Solution {
public:
    int uniquePathsWithObstacles(vector<vector<int>>& obstacleGrid) {
        int m=obstacleGrid.size();
        int n=obstacleGrid[0].size();
        vector<vector<int>>dp(m,vector<int>(n,0));
        int flag=0;
        for(int i=0;i<n;i++){
            if(obstacleGrid[0][i]==1)flag=1;
            if(flag==1)dp[0][i]=0;
            else dp[0][i]=1;
        }
        flag=0;
        for(int i=0;i<m;i++){
            if(obstacleGrid[i][0]==1)flag=1;
            if(flag==1)dp[i][0]=0;
            else dp[i][0]=1;
        }
        for(int i=1;i<m;i++){
            for(int j=1;j<n;j++){
                if(obstacleGrid[i][j]!=1)
                dp[i][j]=dp[i-1][j]+dp[i][j-1];
                else
                dp[i][j]=0;
            }
        }
        return dp[m-1][n-1];
        
    }
};
```

拆出的数近似相等，乘积最大

1.dp数组的含义

dp[i]对i进行拆分，得到的最大乘积为dp[i]

2.递推公式

dp[i]=j×dp[i-j]

这个递推公式可以包含所有拆成奇数个和偶数个的情况。

3.初始化

dp[0]=0

dp[1]=0

dp[2]=1

4.遍历顺序

从i=3开始遍历

j从0到i的一半

dp[i]=max(j*（i-j），j×dp[i-j]，dp[i])；分别对应拆成两个数，拆成多个数，和以前的最大值。

class Solution {
public:
  
    int integerBreak(int n) {
        vector<int> dp(n+1,0);
        dp[0]=0;
        dp[1]=0;
        if(n==2)return 1;
        dp[2]=1;
        for(int i=3;i<=n;i++){
            for(int j=0;j<=i/2;j++){
                dp[i]=max(j*(i-j),max(j*dp[i-j],dp[i]));
            }
        }
        return dp[n];
    }
};
```
