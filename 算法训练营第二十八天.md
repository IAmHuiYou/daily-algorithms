## 我的代码

```
class Solution {
public:
    int lastStoneWeightII(vector<int>& stones) {
        int sum=0;
        for(int i=0;i<stones.size();i++)sum+=stones[i];
        vector<int> ve(sum/2+1,0);
        for(int i=0;i<stones.size();i++){
            for(int j=sum/2;j>=stones[i];j--){
                ve[j]=max(ve[j],ve[j-stones[i]]+stones[i]);
            }
        }
        return sum-2*ve[sum/2];
        
    }
};
```

## 卡的思路

复述一遍思路。

我也感觉回溯可以暴搜出来，但是时间复杂度2^n超时不要太轻松

推导：

 `left组合 - right组合 = target`

`left + right = sum`

 `left = (target + sum)/2` 

其实就是把数字分成两堆，一堆减去另一堆。加法对应的是left。

问题转化成用nums装满容量为x的背包，有几种方法。

需要注意的是，如果不能整除，没有解。如果target大于sum，也没有解。

1.dp数组及下标含义。

dp[i][j]：使用 下标为[0, i]的nums[i]能够凑满j（包括j）这么大容量的包，有`dp[i][j]`种方法。

2.递推公式

`if(nums[i]>j)dp[i][j]=dp[i-1][j];`

`else dp[i][j]=dp[i-1][j]+dp[i-1][j-nums[i]];`

3.初始化

`dp[0][0]`有一种方法，即不放

`dp[0][j]`其中当`dp[0][nums[0]]=1`,其他都是0，装不满或装不下。

`dp[i][0]`如果有物品数值为0，那么有2^n种

否则就只能放0件，也就是一种方法。

4.确定遍历顺序

从上到下从左到右。因为每一个都右上方和左上方推导出来。

不太简单的，很多细节。

## 我的代码

```
class Solution {
public:
    int findTargetSumWays(vector<int>& nums, int target) {
        int sum=0;
        for(int i=0;i<nums.size();i++)sum+=nums[i];
        if(abs(target)>sum)return 0;
        if((target+sum)%2==1)return 0;

        int bagsize=(target+sum)/2;

        //初始化
        vector<vector<int>> dp(nums.size(),vector<int>(bagsize+1,0));
       if(nums[0]<=bagsize)dp[0][nums[0]]=1;
       dp[0][0]=1;
        int zero=0;
        for(int i=0;i<nums.size();i++){
            if(nums[i]==0)zero++;
             dp[i][0]=(int)pow(2.0,zero);
        }

        //递推
        for(int i=1;i<nums.size();i++){
            for(int j=0;j<=bagsize;j++){//对于二维的dp数组，遍历内外次序无所谓
            if(nums[i]>j)dp[i][j]=dp[i-1][j];
               else  dp[i][j]=dp[i-1][j]+dp[i-1][j-nums[i]];
            }
        }
        return dp[nums.size()-1][bagsize];
        
        
    }
};
```

## 卡的思路

复述思路。

此时背包有两个维度，分别是0的个数和1的个数，仍然是01背包。

1.dp数组的含义

二维的dp数组，三个变量，m的个数n的个数

`dp[i][j]`,i个0，j个1，最大背`dp[i][j]`物品。

2.递推公式

chun01背包递推公式：

`dp[j]=max(dp[j],dp[j-weight[i]]+value[i])`

背包的重量其实是0和1的个数

`dp[i-x][j-y]+1`对应`dp[j-weight[i]]+value[i])`

`max(dp[i-x][j-y]+1，dp[i][j])`

3.初始化

`dp[0][0]=0`

其他也全为0，因为不能覆盖掉`dp[i-x][j-y]+1`

4.遍历顺序

对字符串集合进行遍历

再对其中的每个字符遍历，统计0和1的个数。

先遍历物品、再遍历背包。背包倒序遍历。

## 卡的代码

```
class Solution {
public:
    int findMaxForm(vector<string>& strs, int m, int n) {
        vector<vector<int>> dp(m + 1, vector<int> (n + 1, 0)); // 默认初始化0
        for (string str : strs) { // 遍历物品
            int oneNum = 0, zeroNum = 0;
            for (char c : str) {
                if (c == '0') zeroNum++;
                else oneNum++;
            }
            for (int i = m; i >= zeroNum; i--) { // 遍历背包容量且从后向前遍历！
                for (int j = n; j >= oneNum; j--) {
                    dp[i][j] = max(dp[i][j], dp[i - zeroNum][j - oneNum] + 1);
                }
            }
        }
        return dp[m][n];
    }
};
```
