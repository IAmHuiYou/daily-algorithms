## 卡的思路

复述思路。

比较简单的解法是从左边找一个最小值，从右边找一个最大值。

**dp解法：**

1.dp数组的含义

`dp[i][j]`

`dp[i][0]`表示第i天持有股票的最大现金

`dp[i][1]`表示第i天不持有股票的最大现金

最终要求的结果是`max(dp[i][0],dp[i][1])`

2.递推公式

`dp[i][0]=max(dp[i-1][0],-price[i])`

保持前一个持有的状态，或者在第i天买入

`dp[i][1]=max(dp[i-1][1],dp[i-1][0]+price[i])`

保持前一个不持有的状态，或者在第i天卖出

3.初始化

`dp[0][0]=-price[0]`

`dp[0][1]=0`

4.遍历顺序

从前向后

## 我的代码

```
class Solution {
public:
    int maxProfit(vector<int>& prices) {
    vector<vector<int>>dp(prices.size(),vector<int>(2,0));
    dp[0][0]=-prices[0];
    for(int i=1;i<prices.size();i++){
        dp[i][0]=max(dp[i-1][0],-prices[i]);
        dp[i][1]=max(dp[i-1][1],dp[i-1][0]+prices[i]);
    }
    return max(dp[prices.size()-1][0],dp[prices.size()-1][1]);
    }
};
```

## 卡的思路

复述思路

用贪心的话只要从前向后收割正利润即可。

2.递推公式

`dp[i][0]=max(dp[i-1][0],dp[i-1][1]-price[i])`

当天持有股票的最大收益等于 继续持有前一天股票的最大收益 和 之前累积下来的股票的最大收益减去今天买的股票的前 的最大值。

**与上一题的区别：**

上一题``dp[i][0]=max(dp[i-1][0],-price[i])``

本题`dp[i][0]=max(dp[i-1][0],dp[i-1][1]-price[i])`

区别在于上一题因为只能买卖一次，所以当买入的时候手上的钱是0-price，而而本题的钱可以累计，所以是累计的钱减去price

`dp[i][1]=max(dp[i-1][0]+price[i],dp[i-1][1]);`

## 我的代码

```
class Solution {
public:
    int maxProfit(vector<int>& prices) {
        vector<vector<int>>dp(prices.size(),vector<int>(2,0));
        dp[0][0]=-prices[0];
        for(int i=1;i<prices.size();i++){
            dp[i][0]=max(dp[i-1][0],dp[i-1][1]-prices[i]);
            dp[i][1]=max(dp[i-1][1],dp[i-1][0]+prices[i]);

        }
        return dp[prices.size()-1][1];
    }
};
```

## 卡的思路

复述一下思路。

至多买卖两次的意思是，可以买卖两次、一次、0次

1.dp数组的含义

`dp[i][0]`不操作：手头的现金为0

`dp[i][1]`第一次持有

`dp[i][2]`第一次不持有

`dp[i][3]`第二次持有

`dp[i][4]`第二次不持有

2.递推公式

```
`dp[i][0]=dp[i-1][0]`
//第i天第一次持有，可以是前一天持有的状态推导而来，或者是前一天是不操作状态，在i天买入
dp[i][1]=max(dp[i-1][1],dp[i-1][0]-price[i])

//延续前一天不持有的状态，或者在i天卖出
dp[i][2]=max(dp[i-1][2],dp[i-1][1]+price[i])

// 延续前一天第二次持有的状态，或者在第一次卖出的基础上买入
dp[i][3]=max(dp[i-1][3],dp[i][2]-price[i]);

//延续前一天第二次卖出的状态，或者前一天是第二次持有的状态上卖出
dp[i][4]=max(dp[i-1][4],dp[i-1][3]+price[i])
```



3.初始化

`dp[0][0]`无操作，手上现金就是0

`dp[0][1]`第一次买入-price[i]

`dp[0][2]`理解成同一天买卖，手上现金为0

`dp[0][3]`同一天第二次买入-price[i]

`dp[0][4]=0`



4.遍历顺序

从前往后。

最后的结果应该是两次卖出的状态取最大值，因为卖出的肯定比买入的手上现金多的。

此外，因为第二次卖出的状态已经包含了第一次卖出的状态了，所以直接返回第二次卖出的状态即可。

这题 总的来说就是增加了多种状态，这里的“状态”比前面的题目更明显，第一次让我意识到“状态”，它不是动作，因为动作是不确定的，不确定什么时候买，什么时候卖。

## 我的代码

```
class Solution {
public:
    int maxProfit(vector<int>& prices) {
        vector<vector<int>>dp(prices.size(),vector<int>(5,0));
        dp[0][1]=-prices[0];
        dp[0][3]=-prices[0];
        for(int i=1;i<prices.size();i++){
            dp[i][1]=max(dp[i-1][1],-prices[i]);
            dp[i][2]=max(dp[i-1][2],dp[i-1][1]+prices[i]);
            dp[i][3]=max(dp[i-1][3],dp[i-1][2]-prices[i]);
            dp[i][4]=max(dp[i-1][4],dp[i-1][3]+prices[i]);
        }
        return dp[prices.size()-1][4];
    }
};
```
