**学习笔记**

二维dp递推公式

`dp[i][j] = max(dp[i - 1][j], dp[i - 1][j - weight[i]] + value[i]);`

分析：当前层是由上一层推导出。

因此可以把上一层数据拷贝到当前层，直接计算当前层。因此叫滚动数组

动规五部曲

1.`dp[j]`

表示容量为j的背包所能装的最大价值为`dp[j]`

2.递推公式

dp[j]=max(dp[j],dp[j-value[i]]+value[i]);

不放物品i对应的就是从上一层拷贝下来的dp[j]；放物品i，要先减去i的重量，加上i的价值。

观察公式变化，相当于二维dp数组把i部分去掉即可。

3.初始化

dp[0]：背包容量为0所能装的最大价值—>0

其他值因为每次都要取自己和`dp[j-value[i]]+value[i]`的最大值，不能覆盖掉我们推导的结果。

因此初始化为0.

4.遍历顺序

```cpp
for(int i = 0; i < weight.size(); i++) { // 遍历物品
    for(int j = bagWeight; j >= weight[i]; j--) { // 倒序遍历背包容量
        dp[j] = max(dp[j], dp[j - weight[i]] + value[i]);

    }
}
```

为什么要倒序遍历背包？

正序遍历会多次添加一种物品。

## 卡的思路

这题需要注意的是，组合不是排列数，不可以重复。

这里用的是二维的dp数组

1.dp数组及下标含义

使用下标为[0,i]的coin[i]能够凑满j容量的包，有`dp[i][j]`种方法。

2.递推公式

`dp[i][j] = dp[i - 1][j] + dp[i][j - nums[i]]`

也就是不用这种硬币的方案数和用这种硬币的方案数。

3.初始化

`dp[0][j]`上面一行表示只放coin[0]能放的个数，如果能整除，应该是1种，不能就是0

`dp[i][0]`表示容量为0的时候的方案，因为没有重量为0的，所以都只有一种方案，就是不放。即0

4.遍历顺序

二维dp遍历顺序是无所谓的。

## 我的代码

```
class Solution {
public:
    int change(int amount, vector<int>& coins) {
        int bagSize = amount;

        vector<vector<uint64_t>> dp(coins.size(), vector<uint64_t>(bagSize + 1, 0));

        // 初始化最上行
        for (int j = 0; j <= bagSize; j++) {
            if (j % coins[0] == 0) dp[0][j] = 1;
        }
        // 初始化最左列
        for (int i = 0; i < coins.size(); i++) {
            dp[i][0] = 1;
        }
        // 以下遍历顺序行列可以颠倒
        for (int i = 1; i < coins.size(); i++) { // 行，遍历物品
            for (int j = 0; j <= bagSize; j++) { // 列，遍历背包
                if (coins[i] > j) dp[i][j] = dp[i - 1][j]; 
                else dp[i][j] = dp[i - 1][j] +  dp[i][j - coins[i]];
            }
        }
        return dp[coins.size() - 1][bagSize];
    }
```
